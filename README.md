## Running locally

Clone project

```bash
  git clone https://link-para-o-projeto
```

Go into the project directory

```bash
  cd my-project
```

Execute docker

```bash
  docker build
```

Init server

```bash
  docker run app
```

## Running tests

To run the tests, run the following command

```bash
  mvn test
```
